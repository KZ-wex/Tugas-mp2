# LinkedIn UI Clone - Tugas Mobile Programming 2

Repositori ini berisi implementasi antarmuka pengguna (UI) yang terinspirasi dari LinkedIn menggunakan **Flutter**. Project ini dikembangkan sebagai bagian dari tugas mata kuliah Mobile Programming, yang mencakup materi dasar hingga integrasi aset dan data dummy.

## 🚀 Fitur Utama

*   **Beranda (Home Feed)**: Menampilkan postingan dinamis dengan data dummy dari Unsplash API untuk memberikan kesan aplikasi yang hidup dan profesional.
*   **Halaman Profil**: Berfungsi sebagai portofolio yang memuat materi pembelajaran 1 hingga 7.
*   **Navigasi Modern**: Menggunakan `SalomonBottomBar` untuk pengalaman navigasi yang lebih estetik dan interaktif.
*   **Custom Assets**: Integrasi foto profil lokal menggunakan folder `assets`.

## 🛠️ Tech Stack

*   **Framework**: [Flutter](https://flutter.dev/)
*   **Language**: Dart
*   **Database/Backend**: [Firebase](https://firebase.google.com/) (Initialization ready)
*   **Images**: Unsplash API (Network Images) & Local Assets

## ⚙️ Cara Menjalankan Project

1.  **Clone repositori ini**:
    ```bash
    git clone [https://github.com/KZ-wex/Tugas-mp2.git](https://github.com/KZ-wex/Tugas-mp2.git)
    ```
2.  **Masuk ke direktori**:
    ```bash
    cd Tugas-mp2
    ```
3.  **Ambil dependensi**:
    ```bash
    flutter pub get
    ```
4.  **Jalankan aplikasi**:
    ```bash
    flutter run
    ```
